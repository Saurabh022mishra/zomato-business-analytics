 # Zomato Business Analytics Project

## 📌 Overview
This project analyzes the Zomato restaurant dataset to generate business insights for decision-making in the food delivery industry.

## 📊 Objectives
- Perform Descriptive Analytics (EDA)
- Build Predictive Models
- Identify key factors affecting restaurant ratings
- Segment restaurants using clustering

## 📁 Dataset
- zomato.csv
- Country_Code.xlsx

## ⚙️ Methods Used
### Descriptive Analytics
- Geographic distribution
- Cuisine trends
- Pricing vs ratings
- Online delivery impact
- Correlation analysis

### Predictive Analytics
- Linear Regression
- Random Forest
- Gradient Boosting

### Clustering
- K-Means clustering
- PCA visualization

## 📈 Output
- 9 professional visualizations
- Model performance metrics
- Cluster segmentation insights
